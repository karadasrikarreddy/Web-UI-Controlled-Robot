# Web-UI Controlled Robot

## Overview
This project is a Web-UI Controlled Robot that allows users to control a physical robot through a user-friendly web interface. The system includes a hardware interface, communication protocols, and a web application to facilitate control and monitoring of the robot.

## Features
- Real-time control of the robot from any web browser
- Responsive user interface for easy interaction
- Live video feed from the robot's camera 
- Optional autonomous modes

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript (React)
- **Backend:** Node.js, Express
- **Communication:** WebSocket for real-time communication
- **Hardware:** Raspberry Pi, Arduino, or compatible devices

## Getting Started
### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or above)
- [npm](https://www.npmjs.com/) (Node Package Manager)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/karadasrikarreddy/Web-UI-Controlled-Robot.git
   cd Web-UI-Controlled-Robot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Access the web interface at `http://localhost:3000`
